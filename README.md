# Dice_game
To design and implement a dice game where a player competes against the computer. The program should simulate dice rolls, calculate scores, and determine the winner based on a target score.

This project is a simple Dice Rolling Game where the player plays against the computer. Both the player and the computer take turns rolling a dice.
Each dice roll generates a random number between 1 and 6, which is added to the respective player's score. The game continues in turns until one of them reaches the target score (30 points).
Game logic:
• The player rolls the dice by pressing Enter
• The computer automatically rolls the dice
• Scores are updated after each turn
• The first to reach 30 points wins the game
This project demonstrates the use of functions, loops, random number generation, and conditional logic in Python.
